# Get the epishiny Translator object

Returns the package-level
[shiny.i18n::Translator](https://appsilon.github.io/shiny.i18n/reference/Translator.html)
instance, creating it on first use from
[inst/translations/translation.json](https://epicentre-msf.github.io/epishiny/reference/translations/translation.json).

## Usage

``` r
get_epishiny_i18n()
```

## Value

A
[shiny.i18n::Translator](https://appsilon.github.io/shiny.i18n/reference/Translator.html)
object.
