# Initialise or reset the epishiny Translator

Initialise or reset the epishiny Translator

## Usage

``` r
init_epishiny_i18n(lang = "en")
```

## Arguments

- lang:

  Two-letter language code (e.g. `"en"`, `"fr"`).

## Value

The
[shiny.i18n::Translator](https://appsilon.github.io/shiny.i18n/reference/Translator.html)
object (invisibly).
